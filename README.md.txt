# 🌦️ Algeria Coastal Weather Forecast (2022–2026)

This project analyzes and forecasts weather conditions (temperature and rainfall) across Algeria's coastal cities.  
It uses historical data (2022–2024) and Machine Learning models to predict weather for future years like 2026.
